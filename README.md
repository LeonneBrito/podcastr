# :computer: How to run

```bash
# Clone Repository
$ git clone https://github.com/LeonneBrito/podcastr.git
```

```bash
# Install Dependencies
$ yarn

#Run server
$ yarn server

# Run Aplication
$ yarn dev
```
Go to http://localhost:3000/ to see the result.
